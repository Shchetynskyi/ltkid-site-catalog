# SITE-CATALOG

## Тип продукту
Публічний сайт-каталог окулярів.
НЕ є інтернет-магазином.

Відсутні:
- оплата
- кошик
- доставка
- гарантії
- онлайн-замовлення

Усі дії купівлі та консультації відбуваються через менеджера (Messenger / ManyChat).

## Архітектура
- Stack: SvelteKit + TypeScript
- Frontend: read-only
- Дані: Google Sheets (CSV)
- Запис назад у таблицю: відсутній

## Source of Truth
Єдине джерело правди  Google Sheets / CATALOG_V2.

Фронтенд:
- читає CSV
- не змінює структуру
- не залежить від автоматизацій імпорту

## Основна логіка
- Показуються ТІЛЬКИ рядки з Показувати = Так
- Категоризація:
  - ready  якщо є діоптрії виробника
  - frames  якщо немає
- Gender:
  - жіноча / чоловіча / унісекс
  - unisex входить у обидві галереї

## Поточний статус
Система стабільна.
Готова до подальшого розширення (Try-On, AI, підбір).
